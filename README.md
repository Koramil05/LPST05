# The Royal Wedding - PWA Display

Aplikasi display untuk pernikahan dengan fitur:
- Slideshow gambar & video
- Player MP3 dengan playlist
- Running text dengan efek kilauan
- Logo kiri/kanan dengan rotasi otomatis
- Tanggal & waktu real-time
- Panel pengaturan lengkap
- Auto hide controls
- **PWA Ready** - bisa diinstall di HP/Tablet/Android TV
